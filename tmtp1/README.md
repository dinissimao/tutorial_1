# 1. Introduction

O presente trabalho tem como objetivo introduzir os conceitos
fundamentais da programação em **Kotlin** e a organização de projetos
utilizando uma estrutura típica de desenvolvimento baseada em **Maven**.

Este tutorial permite compreender os princípios básicos da linguagem
Kotlin, bem como a estruturação de aplicações simples através de
diferentes exercícios.

------------------------------------------------------------------------

# 2. System Overview

O sistema desenvolvido consiste num conjunto de pequenos módulos
implementados em Kotlin, organizados em diferentes packages.

Componentes principais:

-   Main.kt -- ponto de entrada da aplicação
-   Exercícios básicos (exer_1, exer_2, exer_3)
-   Virtual Library -- pequena aplicação que simula uma biblioteca

------------------------------------------------------------------------

# 3. Architecture and Design

Estrutura principal do projeto:

src/ └── main/ └── kotlin/ └── cm/ ├── exer_1 ├── exer_2 ├── exer_3 └──
virtual_library

Cada package representa um conjunto específico de funcionalidades.

------------------------------------------------------------------------

# 4. Implementation

A implementação foi realizada em **Kotlin**, utilizando uma estrutura
baseada em **Maven**.

Classes principais:

-   Book
-   DigitalBook
-   PhysicalBook
-   Library

Estas classes permitem representar diferentes tipos de livros e gerir
uma pequena biblioteca virtual.
